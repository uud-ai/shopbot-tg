# ShopBot 🛍 — Telegram Shopping Assistant

Личный шоппинг-ассистент в Telegram. Помогает найти товары на Ozon, Wildberries, Яндекс Маркет и AliExpress.

## Переменные окружения (Railway)

| Переменная | Значение |
|---|---|
| `BOT_TOKEN` | Токен от @BotFather |
| `OPENROUTER_KEY` | API ключ с openrouter.ai |

## Команды бота

- `/start` — начать / сбросить историю
- `/clear` — очистить контекст диалога  
- `/help` — справка
